# Lesson 9. Do it. Don't do it. Stop doing it!


## Rules

Simple command - just infinitive.  
Restriction - STOP V-ing  

Express command that you wish other to do.  
Also use for forbidding something (restriction, prohibition, banning).  

Come here!  
Go to hell!  
Don't smoke here.  
Stop smoking here!
